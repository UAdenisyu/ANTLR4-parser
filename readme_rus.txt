Для запуска приложения запустите parse.bat

###

Eсли грамматика языка не была изменена,
то можно(даже нужно) запускать сразу файл main.py

###

Данный 'батник' создаст(или перезапишет) файлы antlr4
на языке Python3.7, необходимые для дальнейшего 
парсинга исходного кода, записанного на вашем языке
программирования в файле program_example.pypas
(предварительно удалив предыдущие, если таковые имеются).
Затем запустит main.py(не изменяем содержимое)

###

!!!Для корректной работы батника нужно, чтоб в текущей
директории находились следующие файлы:

antlr-4.8-complete.jar
main.py
program_example.pypas
Test.g4

###

!!!Также необходимо, чтобы грамматика языка(Test.g4) 
  была записана в нотации antlr4, а также первое
  (и основное синтаксическое с минимальным приоритетом) 
  правило парсера называлось
  'prog'(как в начальном примере грамматики).
  Имя грамматики должно быть 'Test'.
  Название файла с грамматикой должно называться 'Test.g4'.
  Eсли грамматика записана неправильно,
  то batник не сработает.

###

В примере приведена элементарная грамматика языка Test.
Описание:
Первое слово каждой новой строки должно быть test
После слова test должно быть любое слово,
которое состоит из маленьких букв латинского алфавита.
Пробелы и символы новой строки игнорим.
После второго слова ничего не должно быть (ну кроме пробелов и \n)
